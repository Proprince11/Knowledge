## Conventions

- **One topic per file.** Filenames are lowercase-kebab-case.
- **Metadata header** at the top of every file (status, depth, prerequisites, last updated).
- **Structure:** Technical Mechanisms → Application Frameworks → Common Pitfalls → Advanced Resources.
- **Zero fluff.** Graduate-level depth, with formulas, data, code, and primary-source links.

## Disclaimers

- **Security content** (Domain 04) is written for authorized testing, defensive auditing,
  and hardening only. Do not apply techniques to systems you do not own or lack written
  permission to test.
- **Health content** (Domain 05) is educational and is not medical advice. Consult a
  qualified clinician before changing diet, training, or supplementation.
