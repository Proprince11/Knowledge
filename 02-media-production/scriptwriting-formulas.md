# Advanced Scriptwriting Formulas

Story structure is not a creativity straitjacket — it is **a model of how human attention and
emotion respond over time**. The "formulas" (three-act, hero's journey, Save the Cat, Dan
Harmon's circle) are parameterizations of the same dynamics: *desire → obstacle → change*. This
file treats them as interchangeable tools, then drills into the unit that does the work — the
**scene** — and the craft of dialogue and the retention-critical open.

---

## 1. Technical Mechanisms

### 1.1 The universal engine: desire, conflict, change

```
A character WANTS something (goal) → faces an OBSTACLE (conflict) → which forces a CHOICE →
producing CHANGE (a new value state). No want → no story. No obstacle → no drama.
```
- **Want vs. Need:** the *want* (external goal) drives plot; the *need* (internal lack) drives
  the arc. The most resonant endings deliver the need, often by sacrificing the want.
- **Stakes:** what is lost on failure. No stakes = no tension. Escalate them.

### 1.2 Major structural models (one engine, different maps)

| Model | Beats | Best for |
|---|---|---|
| **Three-Act** | Setup → Confrontation → Resolution (two turns) | universal scaffold |
| **Hero's Journey** (Campbell/Vogler) | ordinary world → call → threshold → ordeal → return | myth, transformation |
| **Save the Cat** (Snyder) | 15 beats (opening image → catalyst → midpoint → all-is-lost → finale) | commercial pacing |
| **Story Circle** (Harmon) | You → Need → Go → Search → Find → Take → Return → Change | episodic, short-form |
| **Sequence method** | 8 ~12–15 min sequences | feature pacing in chunks |

> **They're isomorphic.** Harmon's circle is the hero's journey on a ring; Save the Cat is three
> acts with named beats. Learn one deeply; recognize the others as re-labelings.

### 1.3 Midpoint, low point, value shifts

- **Midpoint:** a false victory/defeat or revelation that recontextualizes the goal — turns a
  *reactive* protagonist *proactive*. Kills the "saggy middle."
- **All-is-lost:** maximum pressure forces the internal change the need required.
- **McKee's value charge:** every scene shifts a value (life/death, love/hate) + to − or − to +.
  A scene where nothing changes is a deleted scene.

### 1.4 The scene as atomic unit

```
GOAL (who wants what) → CONFLICT (opposition) → TURN (value flips) → OUT (hook to next)
```
- **Enter late, leave early:** start at the latest moment; cut at the turn.
- **The "but/therefore" test** (Parker & Stone): beats connect with *"but"* or *"therefore,"*
  never *"and then"* — the signature of a plotless list.

### 1.5 Dialogue mechanics

- **Subtext:** the gap between what's said and meant *is* the drama; on-the-nose dialogue kills
  tension.
- **Each line does work:** plot, character, or conflict — ideally all three.
- **Voice differentiation:** cover the names and you still know who speaks.
- **Action over exposition:** dramatize; bury exposition inside conflict.

## 2. Application Frameworks

### 2.1 Short-form / digital structure (YouTube, Reels, TikTok)

```
HOOK (0–5s)      promise/curiosity gap — the "want" the VIEWER adopts
SETUP (fast)     minimal context; raise the stake/question
ESCALATION       value in rising beats; each segment re-earns the next
PAYOFF           satisfy the promise (the "need")
LOOP/CTA         close that points forward (next video / re-watch)
```
The retention curve (`youtube-algorithm-mastery.md` §2.2) is the empirical value charge: dips =
scenes with no turn; spikes = strong turns/payoffs.

### 2.2 Logline & premise test

```
When [INCITING INCIDENT], a [FLAWED PROTAGONIST] must [GOAL/STAKE] or else [CONSEQUENCE].
```
If the logline isn't compelling, no execution saves it. It forces clarity on protagonist, want,
stakes, and irony.

### 2.3 Outline-then-draft workflow

```
1. PREMISE/LOGLINE   one sentence; test it
2. BEAT SHEET        map your model's beats
3. SCENE LIST        one line/scene: goal/conflict/turn; check but-therefore chain
4. DRAFT             write fast and ugly; structure de-risked the story
5. REVISE            cut 10%; sharpen turns; deepen subtext; differentiate voices
```

### 2.4 Genre as a contract

Genre sets expectations (the promise to the audience). Master conventions, then subvert
*deliberately* — fulfilling the emotional promise while surprising on specifics.

### 2.5 Character arc

```
FLAW (the lie believed) → tested by escalating stakes → CRISIS (old lie vs. new truth) →
CHANGE (embraces truth, pays a price) → NEW EQUILIBRIUM
```
External events force the internal choice; the internal change enables the external resolution.

## 3. Common Pitfalls

1. **No clear want/stakes.** Aimless protagonist = no engine.
2. **"And then" plotting.** No causal chain.
3. **On-the-nose dialogue / no subtext.**
4. **Saggy middle.** No midpoint shift.
5. **Exposition dumps** outside conflict.
6. **Late hook** (long-form) loses digital audiences in seconds.
7. **Want without need** (or vice versa).
8. **Undifferentiated voices.**
9. **Stakes that don't escalate** → unearned climax.
10. **Structure worship** — beats without emotional truth.
