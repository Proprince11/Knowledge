# Webflow Asset Blueprints

A spec library of **15 SSS-rank Webflow templates** and **10 ultra-premium e-commerce stores**, each engineered as a $10K+-caliber asset brief.

These are **written specifications** — design systems, page-by-page architecture, interaction logic, and editor guides — intended to be handed to a Webflow developer or used as a build roadmap. They are not buildable Webflow project files; Webflow's `.webflow` format is only writable from inside the Webflow Designer GUI.

## Structure

```
webflow-blueprints/
├── templates/        # 15 Webflow templates (SaaS, agency, luxury, fintech, etc.)
└── stores/           # 10 e-commerce / digital goods storefronts
```

## Naming convention

Files follow `NN_template-slug.md` so they sort in build order.
